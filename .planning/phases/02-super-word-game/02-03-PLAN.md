Internal helper (not exported):
```typescript
function announce(message: string, priority: 'polite' | 'assertive'): void {
  const el = document.getElementById(priority === 'assertive' ? 'game-feedback' : 'game-status')
  if (el) el.textContent = message
}
```

Export these functions with EXACT announcement text from UI-SPEC §Screen Reader Announcement Table:

- `announceLetterCollected(char: string, label: string, collected: number, total: number): void`
  → assertive: `` `Collected letter ${char} from ${label}. ${collected} of ${total} letters found.` ``

- `announceDistractorClicked(label: string): void`
  → assertive: `` `That's not a letter! ${label} is a distractor.` ``

- `announceLetterSelected(char: string, position: number): void`
  → polite: `` `Selected letter ${char} at position ${position}. Tap another letter to swap.` ``

- `announceLettersSwapped(char1: string, char2: string, allLetters: string[]): void`
  → polite: `` `Swapped ${char1} and ${char2}. Letters now: ${allLetters.join(' ')}.` ``

- `announceLetterMoved(char: string, position: number): void`
  → polite: `` `Moved ${char} to position ${position}.` ``

- `announceWrongAnswer(): void`
  → assertive: `"Not quite! Try rearranging the letters."`

- `announceCorrectAnswer(word: string): void`
  → assertive: `` `Correct! You spelled ${word}!` ``

- `announceHint(hint: string): void`
  → polite: `` `Hint: ${hint}` ``

- `announceNextPuzzle(puzzleNum: number, total: number, prompt: string): void`
  → polite: `` `Puzzle ${puzzleNum} of ${total}. ${prompt}` ``

- `announceGameWin(score: number): void`
  → assertive: `` `Congratulations! You solved all 5 puzzles! Final score: ${score}.` ``

- `moveFocus(element: HTMLElement): void`
  → `requestAnimationFrame(() => element.focus())`

- `moveFocusAfterTransition(elementId: string, delayMs: number = 300): void`
  → `setTimeout(() => { const el = document.getElementById(elementId); if (el) requestAnimationFrame(() => el.focus()) }, delayMs)`
  The 300ms default matches the screen transition duration from UI-SPEC.

**IMPORTANT:** Always update `textContent` of the live region element. Never remove/recreate the element, never use `innerHTML`. Screen readers only detect content changes in existing live regions.

**src/super-word/animations.ts** — CSS class-based animation triggers. No JavaScript animation APIs.

Export these functions:

- `animateCollectPop(element: HTMLElement): Promise<void>`
  → Add class `collecting` to the element (triggers `@keyframes collectPop` 650ms from game.css).
  → Return a Promise that resolves on `animationend` event. After resolution, set `element.style.display = 'none'`.
  → If `animationend` doesn't fire within 700ms, resolve anyway (safety fallback).

- `animateItemShake(element: HTMLElement): Promise<void>`
  → Add class `shaking`. Return Promise resolving on `animationend`. On resolve, remove class `shaking`.
  → Fallback: resolve after 500ms.

- `animateTileAppear(element: HTMLElement, delay: number = 0): void`
  → Set `element.style.animationDelay = '${delay}s'`. Add class `appearing`.
  → Listen for `animationend`, then remove class `appearing` and clear `animationDelay`.

- `animateTileWrongShake(elements: HTMLElement[]): Promise<void>`
  → Add class `wrong-shake` to all elements. Return Promise resolving on first `animationend`.
  → On resolve, remove `wrong-shake` from all elements. Fallback: 550ms.

- `animateSolvedLetters(container: HTMLElement): void`
  → Query all `.solved-letter` spans in container. Each already has staggered `animation-delay` from renderer.
  → Add class `solved-pop` to each.

- `setWowMode(container: HTMLElement, enabled: boolean): void`
  → If enabled, `container.classList.add('wow-mode')`. If not, `container.classList.remove('wow-mode')`.

All animation functions listen for `animationend` to clean up classes. Include a timeout fallback for each Promise-returning function in case the animation is interrupted or the element is hidden.

**Helper: refreshGameScreen()**
```typescript
function refreshGameScreen(): void {
  const puzzle = currentPuzzle()
  const state = getState()
  renderScene(puzzle, state, sceneEl)
  renderLetterSlots(state, puzzle, slotsEl)
  renderGameHeader(state, puzzle, state.currentPuzzleIndex, activePuzzles.length)
  setCheckButtonEnabled(state.collectedLetters.length === puzzle.answer.length)
}
```

**Game Flow — InputCallbacks Implementation:**

`onStartGame`:
1. If `startPuzzleParam > 0`: set `gameState = { ...gameState, currentPuzzleIndex: startPuzzleParam }`
2. `refreshGameScreen()`
3. `showScreen('game-screen')`
4. `announceNextPuzzle(gameState.currentPuzzleIndex + 1, activePuzzles.length, currentPuzzle().prompt)`
5. Move focus to first scene item: `moveFocusAfterTransition` → find first `[tabindex="0"]` in `#scene`

`onLetterCollected(item: SceneItem)`:
1. `setState(collectLetter(getState(), item.char!, item.id))`
2. Find the scene item DOM element by `[data-item-id="${item.id}"]`, call `animateCollectPop()` on it
3. `renderLetterSlots(getState(), currentPuzzle(), slotsEl)` — re-render slots
4. Animate the newly added tile: find last `.letter-tile` in slotsEl, call `animateTileAppear()`
5. `announceLetterCollected(item.char!, item.label, getState().collectedLetters.length, currentPuzzle().answer.length)`
6. `showToast('Got it! ✨', 1200)`
7. Update check button: `setCheckButtonEnabled(getState().collectedLetters.length === currentPuzzle().answer.length)`
8. Update letters count in header via `renderGameHeader()`
9. Move scene focus to nearest remaining uncollected item

`onDistractorClicked(item: SceneItem)`:
1. Find DOM element by `[data-item-id="${item.id}"]`, call `animateItemShake()` on its `.item-card`
2. `announceDistractorClicked(item.label)`
3. `showToast('Not a letter! Try another! 🤔', 1500)`

`onTileSelected(index: number)`:
1. `setState(selectTile(getState(), index))`
2. `renderLetterSlots(getState(), currentPuzzle(), slotsEl)` — re-render with selected state
3. If `getState().selectedTileIndex !== null`: `announceLetterSelected(getState().collectedLetters[index].char, index + 1)`
4. If a swap happened (selectTile detected two selected tiles): handle like `onLettersSwapped`

`onLettersSwapped(indexA: number, indexB: number)`:
1. `setState(swapLetters(getState(), indexA, indexB))`
2. `renderLetterSlots(getState(), currentPuzzle(), slotsEl)`
3. Get chars: `const letters = getState().collectedLetters.map(l => l.char)`
4. `announceLettersSwapped(letters[indexA], letters[indexB], letters)`

`onCheckAnswer()`:
1. `const { correct, newState } = checkAnswer(getState(), currentPuzzle())`
2. `setState(newState)`
3. If NOT correct:
   - Query all `.letter-tile` elements in slotsEl, call `animateTileWrongShake(tiles)`
   - `announceWrongAnswer()`
   - `showToast('Not quite — try rearranging! 🔄', 1500)`
4. If correct:
   - `announceCorrectAnswer(currentPuzzle().answer)`
   - Check if this was the last puzzle: `getState().currentPuzzleIndex >= activePuzzles.length - 1`
   - If last puzzle:
     - `renderWinScreen(getState())`
     - `showScreen('win-screen')`
     - `announceGameWin(getState().score)`
     - `moveFocusAfterTransition('replay-btn', 300)`
   - Else:
     - `renderCompleteScreen(currentPuzzle(), getState())`
     - `showScreen('complete-screen')`
     - `animateSolvedLetters(document.getElementById('solved-word')!)`
     - `moveFocusAfterTransition('next-btn', 300)`

`onHintRequested()`:
1. `setState(useHint(getState()))`
2. `announceHint(currentPuzzle().hint)`
3. `showToast('💡 ' + currentPuzzle().hint, 3000)`

`onNextPuzzle()`:
1. `setState(advancePuzzle(getState()))`
2. `refreshGameScreen()`
3. `showScreen('game-screen')`
4. `announceNextPuzzle(getState().currentPuzzleIndex + 1, activePuzzles.length, currentPuzzle().prompt)`
5. Move focus to first scene item

`onPlayAgain()`:
1. `setState(resetGame(getState()))`
2. `showScreen('start-screen')`
3. `moveFocusAfterTransition('start-btn', 300)`

**Initialization:**
1. `setWowMode(gameContainer, wowModeParam)`
2. `setupInput(getState, setState, currentPuzzle, { onStartGame, onLetterCollected, onDistractorClicked, onTileSelected, onLettersSwapped, onCheckAnswer, onHintRequested, onNextPuzzle, onPlayAgain })`
3. `showScreen('start-screen')` (start screen is already `.active` in HTML, but this ensures clean state)
