
**2. Add early theme detection script to `public/index.html` `<head>` (prevents FOUC):**

Insert after the `<link rel="stylesheet">` line, before `</head>`:
```html
<script>
  const theme = localStorage.getItem('theme');
  if (theme) document.documentElement.setAttribute('data-theme', theme);
</script>
```

This is a blocking synchronous script (NOT type="module") that sets `data-theme` before the browser paints. This prevents a flash of wrong theme on page load.

**3. Add `id="games"` to the games section in `public/index.html`:**

Change `<section>` to `<section id="games" class="games-list">` — this gives Plan 02's JS a stable target for dynamic card rendering and applies the flex column layout from Task 1.
